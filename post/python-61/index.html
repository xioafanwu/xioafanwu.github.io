集合类型和操作

  1. 集合的定义
    • set()
    • {}定义
  2. 集合特点
    • 无序
    • 不重复
  3. 集合操作
    • S|T：返回新集合，包含S和T中所有元素
    • S-T：返回新集合，包含在S不在T中元素
    • S&T：返回新集合，S和T中都有的元素
    • S^T：返回新集合，包含S和T不相同元素
    • S<=T：判断S和T的子集关系，返回True或False
    • S|=T:更新集合，包含S和T所有元素
    • S-=T：更新集合
  4. 集合处理方法
    • S.add:增加新元素
    • S.disadd:去除新元素
    • S.remove:去除元素，没有报错
    • S.copy:复制S的一个副本
    • len(S):返回S元素个数
    • x in S:判断S是否有x元素
  5. 应用场景
    • 数据去重
