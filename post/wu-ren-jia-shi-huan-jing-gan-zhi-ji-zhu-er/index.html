交通信号灯与交通标志检测

识别方法

  1. 基于色彩特征
    • 适用于背景简单场景，环境复杂，虚警率高
  2. 基于形状特征
    • 需要建立形状特征规则，不灵活
  3. 基于模板匹配
    • 建立不同样式的交通信号灯模板

交通灯识别系统结构

  1. 图像采集模块
  2. 图像预处理模块
    1. 彩色空间转换
    2. 彩色分割
    3. 噪声去除
    4. 区域标记
  3. 识别模块
    1. 离线训练样本得到分类器
    2. 在线识别
  4. 跟踪模块

识别算法

  1. RGB颜色空间到HSV颜色空间的转化

    • RGB空间三个彩色分量具有相关性，不利于彩色分割
    • HSV空间包含三个分分量（色调、饱和度、明度）

  2. 阈值分割法

    1. 在HSV彩色空间设定阈值，与图像像素HSV各个分量进行比较

    2. 阈值的选取

      1. 大量数据统计
      2. 考虑到不同气候环境、光线下会影响交通信号灯HSV各分量，进行微调

    3. 检测结果

形态学处理

  1. 目的：去除二值化图像的噪声影响，方便感兴趣区域的建立和目标识别后的校验

  2. 方法：腐蚀和膨胀运算

  3. 腐蚀

    • 使用一个宽m,高n的矩形作为模板，像素x置于模板的中心，根据模板的大小，遍历所有被模板覆盖的其他像素，修改像素x的值为所有像素中最小的值。从而将图像外围的突出点加以腐蚀。表现在图像上为黑色区域增大，白色区域减少。

    • 腐蚀结果

  4. 膨胀

    • 膨胀操作与腐蚀操作相反，是将图像的轮廓加以膨胀。操作方法与腐蚀操作类似，也是拿一个矩形模板，对图像的每个像素做遍历处理。不同之处在于修改像素的值不是所有像素中最小的值，而是最大的值。

    • 结果

  5. 形态学处理结果

基于几何形状检测

  • 阈值分割，不满足阈值要求设为0
  • 轮廓查找，计算轮廓点集最外的矩形框
  • 采用矩形框的细长比特征筛选矩形框

基于颜色和形状检测

  • 通过对交通信号灯面积和长宽比的统计，获得交通信号灯灯泡的面积的上限为50，下限为10，交通灯的圆形度≥0.6。
  • 在对图像进行颜色分割处理后，用交通灯的面积和交通灯的圆形度进一步筛选，并将符合阈值要求的交通灯区域采用矩形框形式画出来

深度学习方法介绍

  1. 人工神经网络

    1. 分布式并行信息处理算法数学模型

    2. 节点相互连接

    3. 节点代表一种输出函数——激励函数

    4. 节点之间的连接代表对于通过该连接信号的加权值——权重

    5. 网络本身是一种对自然界某种算法或函数的逼近

    6. 神经网络的训练

  2. 深度学习

    1. 多级非线性变换层级机器学习方法
    2. 多种深度学习框架
      1. 深度神经网络
      2. 卷积神经网络
      3. 深度置信网络

  3. 卷积神经网络（CNN）

    1. 构成

    2. 卷积层

      1. 卷积核大小

      2. 卷积核数量

        1. 一个卷积核提取某一种特征
        2. 多个卷积核提取多重特征
        3. 不同卷积核将提取到不同的特征数据

      3. 步长

        1. 卷积核在输入数据上每次滑动的像素数

      4. 补零

        1. 由于卷积核的大小不一定被输入数据的维度大小所整除，因此卷积核可能不能完全覆盖边界元素，采用补零操作防止信息丢失。

    3. 池化层

      1. 通常在两个相邻的卷积层之间插入池化层操作
      2. 减少参数量，防止过拟合；
      3. 保持局部线性变换的不变性

    4. 全连接层

      1. 卷积层提取了细化的特征，细到人眼看不出来的程度，全连接层就是将学习到的特征整合起来，形成可以实现分类的数据。

前方车辆检测

前车车辆检测方法

  1. 基于视觉
    1. 受雨雾天气影响，对距离判断较弱
  2. 基于雷达的检测方法
    1. 能够判断距离，可选波段有限
  3. 基于红外传感器的检测方法
  4. 抗干扰能力强，对雨雾穿透性弱，成本高
  5. 基于传感器融合的检测方法
  6. 可适应多重环境，准确率高

视觉与二维激光雷达融合检测

  1. 二维激光雷达特点

    • 快速获取扫描平面中的距离信息
    • 获得障碍物在扫描平面中的外轮廓
    • 不受光照条件的影响
    • 无法直接获得障碍物的形状、纹理信息等特征

  2. 机器视觉特点

    • 提供更为丰富的平面信息
    • 容易受光照条件的影响

  3. 摄像机与雷达联合标定

    1. 空间数据融合

      1. 提取标定物在单线激光雷达和图像上的对应特征点来进行摄像机外部参数的标定
      2. 摄像机的外部参数通过约束方程求解后，激光雷达、图像的相对关系完全确定，因此激光雷达扫描点可以通过摄像机模型投影至图像像素坐标系上
      3. 激光雷达外参标定+相机内参标定+4组以上像素雷达电镀，求解线性方程组得到旋转矩阵和平移矩阵

    2. 时间数据融合

      1. 给不同的传感器授予不同的GPS时间，将时间变量作为一个同步参数处理
      2. 使用多线程技术和数据双缓存技术，解决传感器数据滞后问题

    3. 数据融合结果

  4. 路面车辆检测跟踪系统结构

    1. 数据融合模块

    2. 激光雷达模块

      1. 聚类
        1. 目的：将物理或者抽象的对象集合分成有类似的对象组成的多个类
        2. 常见聚类方法
          1. 基于划分的方法：K-means算法
          2. 基于密度的方法：DBSCAN算法
            1. 可以在带有“噪音”的空间数据库中发现任意形状的聚类
        3. 激光雷达的数据特点
          1. 每个角度上只有一个扫描点，而且扫描点顺序排列
          2. 激光雷达的扫描点根据行驶环境的变化而变化，而且聚类的数目不确定，需要根据实际行驶环境情况进行数据聚类
          3. 激光雷达存在较多的噪声点和零散点，需要对这些点进行去噪声和精简处理
          4. 激光雷达的数据密度会随着扫描的距离变化而变化，距离激光雷达原点越近数据密度越高，越远数据密度越低
        4. 激光雷达对车辆特征的扫描主要的几种模型
          1. L模型，当前方车辆在激光雷达的右前方时，扫描点模型表现为L模型
          2. I模型，当前方车辆在激光雷达正前方时，扫描点模型表现为I模型
          3. inv-L模型，当前方车辆在激光雷达的左前方时，扫描点模型表现为inv-L模型

    3. 分类器训练模块

      1. 包括训练数据采集（车辆样本库的建立）、特征提取和分类器训练
      2. 训练数据包括正样本和负样本，其中正样本在采集图片中通过手工定标方法得到，负样本可以在不包含目标的相关背景图片样本中随机抽取或者利用已经完成学习的分类器通过Bootstrapping的方法得到

    4. 视觉模块

      1. 用图像丰富的信息与激光雷达提取的目标位置产生感兴趣区域
      2. 形成候选目标假设校验子窗口
      3. 在子窗口中运用学习后的分类器进行金字塔式的穷尽搜索
      4. 完成目标检测后将结果送至特征融合层与激光雷达提取的特征融合得到目标的位置参数

    5. 跟踪与输出模块

  5. 基于Haar+Adaboost的路面车辆检测

    1. Boosting算法

      • 将多个弱分类器（weakly learnable classifier），通过集成的方法，组合成强学习分类器

    2. Adaboost（Adaptive Boosting，自适应增强算法）

      1. 根据一定规则自动调整样本的权值以及弱分类器的权值

    3. Haar特征

      1. Haar特征模板在图片内以“任意”尺寸“任意”放置，每一种形态为一个特征，找
        出图片所有特征，是进行弱分类训练的基础

    4. 训练流程图

车载毫米波雷达与机器视觉融合

摄像机和毫米波雷达联合标定

  1. 毫米波雷达检测范围

    • 相对距离大于10m时，可以完全覆盖自车车道上的目标
    • 大于30m以后，可以探测旁道上的目标
    • 大于50m以后，可以完全覆盖包括自车车道和左右旁道上的目标

  2. 涉及的坐标系转换

    1. 毫米波局部坐标系、摄像机局部坐标系、车体局部坐标系

    2. 毫米波局部坐标系与车体局部坐标系

      1. 毫米波雷达安装于车头正中间位置，水平居中
      2. 都属于笛卡儿坐标系且单位一致
      3. 只需考虑平移变换

    3. 摄像机局部坐标系与车体局部坐标系

  3. 车辆检测算法设计

    1. 利用毫米波雷达检测到车辆的大体位置，将检测到的车辆坐标投影变换到图像上，确定车辆感兴趣区域
    2. 选用Adaboost分类算法对车辆模型的类Haar特征进行训练，快速得到车辆的粗分类结果，即获取车辆感兴趣区域内疑似车辆候选块
    3. 利用支持向量机（SVM）对HOG特征进行训练，实现车辆的有效检测
