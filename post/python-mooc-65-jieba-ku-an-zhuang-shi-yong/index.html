python mooc 6.5——jieba库安装使用

2019-12-02 python
  1. 安装命令行输入 pip install jieba
  2. 使用
    • jieba.lcut(s):精确模式，返回一个列表类型的分词结果，没有冗余
    • jieba.lct(s.cut_all=True):返回一个列表类型的数据，存在冗余
    • jieba.lct_for_search(s):搜索引擎模式，返回一个列表类的分词结果，存在冗余
