什么是机器学习

  1. Tom Mitchell定义

    一个程序被认为能从经验E中学习，解决任务T，达到性能度量值P，当且仅当，有了经验E后，经过P评判，程序在处理T时的性能有所提升

  2. 机器学习分类

    1. 监督学习-Supervised Learning
    2. 非监督学习-Unsupervised Learning

监督学习

  1. 回归问题，试图预测一个连续的输出值
  2. 分类问题：预测输出离散值
  3. 数据集给定了属性或者标签

非监督学习

  1. 只给出数据集，数据没有属性和标签
  2. 聚类算法：eg:搜索引擎
  3. 非监督学习应用
    1. 组织大型数据中心
    2. 社交网络分析
    3. 市场分隔分析
    4. 天文数据分析
  4. 鸡尾酒宴会问题
    1. 分离音频
