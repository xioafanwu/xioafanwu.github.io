python mooc 6.2——序列类型及操作

2019-11-30 2 min read # python
  1. 类型
    • 元组类型
    • 列表类型
  2. 处理函数和方法
    • x in s:判断x是否在s中
    • x not in s
    • s+t:连接两个序列
    • s*n:将s复制n次
    • s[i]:返回s中的第i个元素
    • s[i:j]/s[i:j:k]:切片，返回s序列第i到j以k为步长的子序列
    • len(s):返回长度
    • min(s):返回序列最小元素
    • max(s):返回最大元素
    • s.index(x)/s.index(x,i,j):返回从第i到j第一次出现元素x的位置
  3. 元组类型
    • 特点
      • 不能被修改
      • 使用()或tuple()创建
      • 也可以不用()
  4. 列表类型
    • 特点
      • 创建后可以修改
      • 使用[]或者list创建
      • 元素类型可以不同
    • 操作函数和方法
      • ls[i]=x:替换列表ls第i元素
      • ls[i:j:k]=It:用列表替换
      • del ls[i]:删除列表中第i元素
      • ls +=lt:更新列表，将列表lt加载ls中
      • ls*=n:更新列表ls，其元素重复n次
      • ls.append(x):在列表ls后面加一个元素x
      • ls.clear:清除所有元素
      • ls.copy:复制一个列表
      • ls.insert(i,x):在第i个位置加一个元素x
      • ls.pop(i):将列表ls中出现的第一个元素x删除
      • ls.remove:将列表ls中出现的第一个元素x删除
      • ls.reserve:将列表ls中元素反转
